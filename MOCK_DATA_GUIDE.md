## 🔍 风险检测逻辑

### 高风险关键词（触发拦截）
- 换汇、JD、拆单、代收款
- 洗钱、诈骗、黑钱
- 帮信、非法
- 缅甸、柬埔寨

### 中风险关键词（触发警告）
- 跨境、大额、转账
- 频繁、异常、可疑

### 评分规则
- **2个以上高风险词**：90-100分 → HALT（拦截）
- **1个高风险词**：75-90分 → WARN（警告）
- **2个以上中风险词**：50-70分 → WARN（警告）
- **其他情况**：15-30分 → ALLOW（允许）

## 📝 技术细节

### 模拟数据生成逻辑
位置：`src/app/api/audit/route.ts`

特点：
- 智能关键词匹配
- 动态评分计算
- 完整的审计报告生成
- 符合真实 API 的数据结构

### 数据结构
```typescript
{
  score: number,              // 0-100 风险评分
  mode: 'NORMAL' | 'AUDIT',   // 审计模式
  decision: 'ALLOW' | 'WARN' | 'HALT',  // 决策
  reason: string,             // 风险原因
  thought_chain: string[],    // 思考链
  ai_response: string         // AI 完整回复
}
```
