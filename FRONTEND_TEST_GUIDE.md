## 🐛 常见问题

### 问题1：页面打不开
**解决**：
```bash
# 检查服务器是否运行
# 如果没有运行，启动它
npm run dev
```

### 问题2：显示"服务异常"
**原因**：这是初始状态，正常的  
**解决**：输入任意消息即可

### 问题3：消息不显示
**解决**：
1. 打开浏览器控制台（F12）
2. 查看 Console 标签
3. 检查是否有错误信息

### 问题4：熔断动画不触发
**检查**：
1. 确保输入了高风险关键词
2. 查看风险评分是否 ≥80
3. 检查浏览器控制台
